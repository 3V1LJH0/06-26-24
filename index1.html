Introduction

Today we were taught about the basics and fundamental tags of HTML. HTML stands for Hypertext Markup Language.An HTML file consists of Document type tag, an html open and closing tag, head tags, and body tags. A Document type tag signifies the extension of file type of the document. HTML tags denotes that the file is indeed an HTML file. The Head tags consist of relevant information about the document such as the document file name, main title page name and its parameters. The Body tags consist all of the text related or the main worspace of HTML. This is where all of the information is allocated as well as the styles.

Paragraph(p) tags contains strings of texts and may also contain other HTML tags as a nested query

(hr) tag means Horizontal Rules, this tag creates a horizontal line that varies on where it is declared or positioned.
